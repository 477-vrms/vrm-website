Project Overview:

Robotic arms and hands that are controlled by a VR headset and controllers. The end product should be able to mimic movements similar to a human arm.

Project Specific Success Criteria:

  1. PSSC #1: An ability to transmit data between the raspberry pi and the microcontroller through a wired connection.
  2. PSSC #2: An ability to control the servos to reach a position to +/- 5 degrees on the robot arm.
  3. PSSC #3: An ability to indicate when it loses connection with the server due to inactivity
  4. PSSC #4: An ability to transmit data from the server to a Raspberry Pi wireless with a latency of less than one second.
  5. PSSC #5: An ability to compute motor angles in Unity and send that data to the server with a latency of less than a half of a second.
